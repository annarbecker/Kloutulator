# Kloutulator
##### Klout Score App | May 10, 2016

#### By Anna Becker & Brad London

## Description

App that finds a user via twitter and returns their Klout score and using Klout API. This is an Epicodus project for practicing Android development concepts. 

## Setup/Installation Requirements
You will need the following programs installed on your computer.
* Android Studio
* Java JDK 8+
* Android SDK

### To Run Application
* In a terminal window, navigate to ~/AndroidStudioProjects
* Run `git clone https://github.com/annarbecker/SociaLite.git`
* Navigate to ~/AndroidStudioProjects/SociaLite
* Run on either an emulator or an Android OS Device connected to a computer

##### To set up an emulator
* Select Run > Run 'app'
* Click 'Create New Emulator'
* Select the device you would like to emulate (Recommended: Nexus 6)
* Select the API level you would like to run - click 'Download' if not available (Recommended: Marshmallow - ABI: x86)
* Click 'Finish' and allow Emulator to run

## Technologies Used

Java, Android Studio, ButterKnife, Firebase, Klout API
